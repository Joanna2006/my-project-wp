FarmBridge App

1. Project Overview  
FarmBridge is a web-based application developed to connect farmers directly with restaurants. The platform allows restaurants to order fresh fruits and vegetables from farmers without involving middlemen. This helps farmers get better prices and ensures fresh delivery to restaurants. The application provides a simple and easy-to-use interface for users.

2. Architecture / Design of the Project  
The project follows a basic web application structure.

Frontend: Developed using HTML, CSS, and JavaScript to design the interface and handle user interaction.  
Backend: JavaScript  
Database: SQLite  

Modules:

- Home Module  
  Displays the main page with categories such as fruits and vegetables.

- Product Module  
  Shows items like apple, mango, tomato, etc.

- Order Module  
  Allows users to select products and place orders.

- UI Module  
  Handles layout, styling, and responsiveness using CSS.

3. APIs Used / Integrated  
No external APIs were used in this project. The application is a static implementation.
